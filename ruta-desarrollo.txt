============================================================
  NOTAS TÉCNICAS
============================================================

  - El backend usa Node.js nativo --watch, no requiere nodemon
  - La BD se levanta con: docker compose up -d
  - Backend en puerto 3000, frontend en 5173 (dev) o 8080 (prod)
  - El .env tiene DB_PORT=5433, docker-compose usa 5434:5432
  - Las imágenes se almacenan en backend/uploads/
  - Los tokens JWT expiran en 7 días
  - No hay pruebas automatizadas configuradas (el comando test
    usa node --test)
